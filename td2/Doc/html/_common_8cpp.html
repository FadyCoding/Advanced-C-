Creates a standard (for the app!) explosion.

Warning
This is not thread safe (as it uses a static variable).
Parameters
posPosition of the explision
soundSamplePathPath of the sound file
maxRadiusRadius the explosion will have at the end.
growingSpeedAmount the radius will grow per second. If <=0, then the value will be "maxRadius * 2".
maxForceMaximum force (at the beginning) the explosion applies to surrounding objects. If <=0, then the value will be "growingSpeed * 2".
maxDamageMaximum thermal (WeaponEnergy::THERMAL) damage (at the beginning) the explosion applies to surrounding objects. If <=0, then the value will be "maxForce / 10000".
