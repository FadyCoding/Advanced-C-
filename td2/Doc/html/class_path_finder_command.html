Method called back by GamePathFinder when the PathFinder was modified by an external command (ex: added an obstacle) BEFORE the execution (onPathFinderReady()).

Returns
true if the path finder should delete this command (= NOT execute it), false if the command should still be processed.
Warning
The call is in the GamePathFinder thread.
