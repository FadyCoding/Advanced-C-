Detailed Description

Runnable path finder (ie.

functor to run in a thread), which accepts a queue of commands to be processed. Commands are treated according to their priority and age (time they've been waiting in the queue).

Gets the internal PathFinder object, to be used synchroneously BEFORE the thread has been started (Not thread safe), or using Lock class.

Thread safety of read/write operations on the PathFinder are left to the caller.
The caller must call triggerGraphModified() once the write operations on the PathFinder are done.
