osg::Vec3f PathFinder::addObstacle ( const osg::Vec3f &  point,
bool  temporary = true 
)
inline

Remove all graph connections from and to the point closest to the given one (projected on the floor).

Temporary obstacles are much faster to add and remove, but they don't speed up the computation (they actually don't remove vertices but mark them with a huge distance). Given point is rounded to closest graph node.

Returns
The point used (corresponds to a graph point), which is the closest from the given one
